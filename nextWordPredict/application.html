Word Prediction

This Shiny App has been made for predicting the next word using input text entered. Natural language processing (NLP) techniques are used for the prediction. The training data used for building the predictive model is from https://d396qusza40orc.cloudfront.net/dsscapstone/dataset/Coursera-SwiftKey.zip

Strategy

I have used:

  • N-grams based approachy and have used Stupid Back-off

  • NGramTokenizer of the RWeka package to make n-grams (n = 1, 2, 3, 4, 5)

  • Computed frequencies from n-grams vectors which is the basis for the prediction algorithms
